循环引用修复指南
==============

1. 删除ViewMainCode.pas中的前向声明:

在ViewMainCode.pas中，删除第11-13行：
```
// HelperLanguage单元中TLanguageManager类型的外部声明
type
  TLanguageManager = class;
```

2. 在ViewMainCode.pas的implementation部分明确声明外部变量:
修改implementation部分为：
```
implementation

uses
  HelperLanguage;

{$R *.dfm}

// 声明外部变量
var
  LanguageManager: TLanguageManager;
```

3. 删除重复的InitLanguageStrings方法:
找到第2874-2881行，注释或删除该部分：
```
// 以下保留兼容旧代码的方法，但实际使用新的语言管理器
procedure TForm1.InitLanguageStrings;
begin
  // 只用于兼容旧的接口
  SwitchToLanguageCode(LanguageManager.CurrentLanguage);
end;
```

4. 删除重复的SetLanguage方法:
找到第2882-2887行，注释或删除该部分：
```
// 根据枚举设置语言
procedure TForm1.SetLanguage(Language: TAppLanguage);
begin
  // 转换为语言代码
  SwitchToLanguageCode(GetLanguageCodeByEnum(Language));
end;
```

5. 确保模块之间的正确依赖顺序:
- UtilsTypes不引用任何业务单元
- ModelLanguage引用UtilsTypes
- HelperLanguage引用ModelLanguage和UtilsTypes
- ViewMainCode在implementation部分引用HelperLanguage
