# SECURITY.md

## Supported Versions

| Version | Supported |
|---------|-----------|
| 2.0.x   | ✅ Yes    |
| < 2.0   | ❌ No     |

## Reporting a Vulnerability

Please report security vulnerabilities **privately** — do not open a public issue.

**Email**: fuyi.it@live.cn

Include:
- Affected version
- Steps to reproduce
- Expected vs actual behavior
- Suggested fix (if any)

We aim to acknowledge within 3 business days and provide a fix timeline within 7 business days.
