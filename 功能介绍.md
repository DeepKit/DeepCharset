# TransSuccess 功能介绍

## 概述

TransSuccess 是一款专业的文件编码转换工具，专为开发人员、文本编辑者和需要处理多语言文件的用户设计。它能够准确检测文件编码并将其转换为目标编码，支持批量处理，让您的文件编码管理变得轻松高效。

## 核心功能

### 1. 精确的编码检测

TransSuccess 采用先进的编码检测算法，能够准确识别以下编码类型：

- **UTF-8（带 BOM 和不带 BOM）**：最常用的 Unicode 编码方式，适用于多语言环境
- **UTF-16 LE/BE**：支持 Little Endian 和 Big Endian 两种字节序的 UTF-16 编码
- **ANSI/ASCII**：传统的单字节编码，适用于英文和西欧语言
- **GB2312/GBK/GB18030**：中文简体编码
- **Big5**：中文繁体编码
- **其他常见编码**：如 Shift-JIS（日文）、EUC-KR（韩文）等

### 2. 灵活的编码转换

支持将文件从任意支持的源编码转换为以下目标编码：

- **UTF-8（带 BOM 或不带 BOM）**
- **UTF-16 LE/BE**
- **ANSI/ASCII**
- **GB2312/GBK**
- **Big5**

您可以选择是否在转换后的文件中添加 BOM（字节顺序标记），以满足不同应用程序的需求。

### 3. 批量处理能力

- **文件夹批处理**：一次性转换整个文件夹中的文件
- **多文件选择**：可选择多个文件进行批量转换
- **文件类型过滤**：可按扩展名筛选要处理的文件
- **并行处理**：利用多核处理器加速批量转换过程

### 4. 安全可靠

- **自动备份**：转换前自动创建原文件备份，确保数据安全
- **错误恢复**：转换失败时可从备份恢复
- **详细日志**：记录转换过程中的每一步操作，便于追踪问题

### 5. 用户友好界面

- **直观的文件浏览器**：轻松选择要转换的文件和文件夹
- **编码树形视图**：清晰展示可用的编码选项
- **实时日志显示**：查看转换进度和结果
- **文件预览**：转换前后可预览文件内容，确保转换正确

### 6. 高级功能

- **递归子目录处理**：可选择是否包含子目录中的文件
- **智能编码检测**：自动识别文件编码，减少手动选择
- **批处理命令生成**：可生成批处理脚本，用于自动化处理

## 技术特点

- **高性能实现**：采用 Delphi 开发，性能卓越
- **低资源占用**：即使处理大文件也保持较低的内存占用
- **无需安装**：绿色软件，无需安装即可使用
- **兼容性好**：支持 Windows 7/8/10/11 系统

## 适用场景

- **开发人员**：处理不同编码的源代码文件
- **网站管理员**：转换网页文件编码，确保正确显示
- **翻译人员**：处理多语言文档
- **数据分析师**：清理和标准化文本数据
- **系统管理员**：批量转换系统文件编码

TransSuccess 是您处理文件编码问题的得力助手，无论是单个文件的快速转换，还是成千上万文件的批量处理，都能轻松应对。
